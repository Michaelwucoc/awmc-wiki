# 授权
本Bot的舞萌功能需要授权。授权范围为群。

:::danger 未授权提示
当你在未经报备的情况下擅自拉群，机器人不会提供舞萌相关功能的服务。
:::

## 免费授权获取
你可以通过以下方式得到免费授权
- 机厅群，官方机厅玩家群，需要有官方字样
![](/images/official_arcade_example.png)
- 本地群，有一定规模的，主要群体为本地玩家的群聊
![](/images/local_arcade_example.png)

如果你的群符合上述条件，请[发送邮件](mailto:awmc@awmc.cc)给我们，按照如下要求填写:

> 主题: 群号 + "机厅授权申请"
> 
> 在内容中 说明机厅地址+机厅群信息界面的截图。

## 付费获取授权
如果是个人爱好群或者是音游同好群，请前往 https://ifdian.net/a/AWMC_TEAM?tab=shop 购买群组授权。
